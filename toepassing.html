Apps

Apps, ofwel applicaties zijn programma’s die bedoeld zijn voor de gebruiker. Zo’n applicatie word gedraaid op een besturingssysteem. Sommige apps kunnen op verschillende besturingssystemen gedraaid worden. We noemen de app dan multiplatform. Adobe Photoshop kan bijvoorbeeld wel gerund worden op windows en macOs, maar niet op chromeOs, android en iOS. Voorbeelden van Apps zijn bijvoorbeeld schrijf applicaties (zoals word), media applicaties en webapplicaties. Webapplicaties worden gerund op een webbrowser. De data hiervan staat opgeslagen op een webserver. Zolang je besturingssysteem een webbrowser kan runnen, kun je webapplicaties zoals gmail gebruiken.

API's

Een API (Application programming interface) is een verzameling aan data, waarmee computerprogramma's met elkaar of andere onderdelen kunnen communiceren. Op deze manier hoeven programma's niet altijd alles zelf te weten, maar kunnen ze informatie opvragen. Een voorbeeld hiervan is hoe niet ieder tekenprogramma zelf hoeft te weten hoe je een printer aanstuurt. In plaats hiervan gebruiken ze een API, waarin deze informatie staat en gedeeld kan worden met zowel Word, Adobe Photoshop en Microsoft Paint. Op deze manier kunnen app (en site) ontwikkelaars ook via API’s toegang krijgen tot andere data. Je zou door middel van een API ook zelf een weer-app kunnen programmeren. Hiermee kun je de data van het weer via de API ontvangen en hoef je niet zelf buiten 24/7 met een temperatuurmeter te staan!
